# ALCASAR Auto Login

Extension Chrome pour le portail ALCASAR de laplateforme.io

## Installation

1. Aller sur `chrome://extensions/`
2. Activer le "Mode développeur"
3. Cliquer sur "Charger l'extension non empaquetée"
4. Sélectionner le dossier de l'extension

## Configuration

1. Cliquer sur l'icône de l'extension dans la barre d'outils Chrome
2. Cliquer sur l'icône ⚙️ (engrenage) pour accéder aux paramètres
3. Entrer l'adresse email et le mot de passe
4. Les identifiants seront sauvegardés automatiquement

## Utilisation

Une fois les identifiants configurés, cliquer sur le bouton "Login" pour se connecter automatiquement au portail ALCASAR.
