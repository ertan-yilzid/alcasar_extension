# ALCASAR Auto Login

Chrome extension for ALCASAR portal at laplateforme.io

## Installation

1. Go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the extension folder

## Configuration

1. Click the extension icon in the Chrome toolbar
2. Click the ⚙️ (gear) icon to access settings
3. Enter your email and password
4. Credentials will be saved automatically

## Usage

Once your credentials are configured, click the "Login" button to automatically connect to the ALCASAR portal.
